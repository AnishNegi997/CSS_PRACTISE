In every flag design, I used these key `position` properties to control the elements:

1. **`position: relative;`** ‚Äì Adjusts an element‚Äôs position relative to its normal flow. I used this to create the base for other elements.
2. **`position: absolute;`** ‚Äì Positions elements relative to the nearest positioned ancestor (like `relative` or `fixed`). This is great for placing specific parts of the flag in a controlled way.
3. **`position: fixed;`** ‚Äì Keeps an element in a fixed position on the screen, regardless of scrolling. Not used directly in flags, but a good tool for UI elements.
4. **`position: sticky;`** ‚Äì Makes elements stick to their container while scrolling. I‚Äôve used this for some flags‚Äô design adjustments!
